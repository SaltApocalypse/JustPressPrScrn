# JustPressPrScrn

给群友瞎写的截屏工具。

## 适用平台

Windows

## 使用方法

- 安装后启动，按下`print_screen`键即可截图。
- 启动后会创建一个托盘，右键启用菜单，`设置`、`退出`都在那里。

## 二次开发

- 开发版本 python=3.11
- 第三方库 `pip install dearpygui loguru keyboard pillow pystray`
